# NotPaylasimSistemi
Üniversite Öğrencilerine Özel Ders Notu Paylaşma WebScripti

- CLAUDE AI kullanılarak yapılmıştır.
- Projeyi kendinizi geliştirmek veya notlar kaydetmek için kullanabilirsiniz.
- Projeyi kendiniz yayınlayacaksanız lisansı kendi dilinize çevirip okumayı unutmayın.



  Installation / Kurulum:

  1- NOTsitesi.rar dosyasını indirin ve açın
  2- notepaylas klasörünün içindeki tüm dosyaları kendi htdocs dosyanıza taşıyın.
  3- notepaylas klasörü içerisindeki database.sql dosyasını localhost/phpmyadmin kısmına gelip import edin.
